# githubUserRepos

## Building project

- Clone repository:
```
git clone https://github.com/ljaworski/githubUserRepos.git
```
- Download dependencies
```
cd githubUserRepos
yarn
```
- Start expo project
```
yarn start
```
- Select connection type to 'Tunnel' in Metro bundler and:
  - scan QR code from Expo app on Android
  - scan QR code from Camera app on iOS (Expo app required)
  
## Tests

To run tests type

```
yarn test
```
